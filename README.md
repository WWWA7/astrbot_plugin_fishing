## 🎣 钓鱼游戏命令手册 🎣

提示：命令中的 [ID] 表示必填参数，<> 表示可选参数。

### 🎣 基础与核心玩法

| 命令 | 描述 |
|---|---|
| `/注册` | 注册用户 |
| `/钓鱼` | 进行一次钓鱼 |
| `/签到` | 每日签到 |
| `/自动钓鱼` | 开启/关闭自动钓鱼 |
| `/钓鱼区域` | 查看钓鱼的区域 |
| `/钓鱼记录` | 查看钓鱼记录 |

### 🎒 背包与资产管理

| 命令 | 描述 |
|---|---|
| `/鱼塘` | 查看用户鱼塘内的鱼 |
| `/鱼塘容量` | 查看鱼塘容量 |
| `/升级鱼塘` | 升级鱼塘容量 |
| `/背包` | 查看用户所有装备 |
| `/鱼竿` | 查看用户鱼竿信息 |
| `/鱼饵` | 查看用户鱼饵信息 |
| `/饰品` | 查看用户饰品信息 |
| `/使用鱼竿 [ID]` | 使用鱼竿 |
| `/精炼鱼竿 [ID]` | 精炼鱼竿 |
| `/使用鱼饵 [ID]` | 使用鱼饵 |
| `/使用饰品 [ID]` | 使用饰品 |
| `/精炼饰品 [ID]` | 精炼饰品 |
| `/精炼帮助` | 查看精炼系统详细说明 |
| `/金币` | 查看用户金币信息 |
| `/高级货币` | 查看用户高级货币 |

### 🛒 商店与市场

| 命令 | 描述 |
|---|---|
| `/全部卖出` | 卖出用户所有鱼 |
| `/保留卖出` | 卖出用户鱼，保留每种一条 |
| `/出售稀有度 [1-5]` | 按稀有度出售鱼 |
| `/出售鱼竿 [ID]` | 出售鱼竿 |
| `/出售所有鱼竿` | 出售所有非五星的鱼竿 |
| `/出售饰品 [ID]` | 出售饰品 |
| `/出售所有饰品` | 出售所有非五星的饰品 |
| `/商店` | 查看商店信息 |
| `/购买鱼竿 [ID]` | 购买鱼竿 |
| `/购买鱼饵 [ID]` | 购买鱼饵 |
| `/市场` | 查看市场 |
| `/上架鱼竿 [ID]` | 上架鱼竿到市场 |
| `/上架饰品 [ID]` | 上架饰品到市场 |
| `/购买 [ID]` | 购买市场上的物品 |
| `/我的上架` | 查看我的上架商品 |
| `/下架 [ID]` | 下架指定商品 |

## 🔧 安装方法

1. 将此插件放入 `data/plugins/` 目录下
2. 重启 AstrBot 或使用插件管理命令加载插件

## ❗注意事项

- 首次使用请先注册
- 钓鱼需要消耗金币，可通过签到获取初始金币
- 钓鱼有冷却时间限制
- 使用高级鱼竿和特殊鱼饵可提高稀有鱼类的获取几率
